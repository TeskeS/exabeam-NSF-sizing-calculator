Average Log Size

To get the average byte size per log, follow these steps:

1. Open Grafana on the XM/Platform Manager

Remote access into the XM/platform manager.

Open a browser and pull up:

https://localhost:3000

2. Open Dashboards

In Grafana, select Dashboards from the left navigation menu.

Select Dashboards from the left-side Grafana menu.

3. Select the Mediator Dashboard

Under the Pipeline folder, select Mediator. If your Grafana doesn’t look like the screenshot, you can search “mediator” in the search bar to locate the dashboard.

Open the Mediator dashboard under Pipeline.

4. Choose a Timeframe

Choose the timeframe you want to use for the metric. I recommend a metric timeframe of at least 2 weeks, if not greater.

Use the Grafana time picker to select the desired timeframe.

5. Locate Average Log Size

The Mediator: Processing widget will show the Average Log Size in Bytes for the timeframe of your choosing.

Use the Average Log Size Bytes value from the Mediator: Processing widget.
If your dashboard doesn’t have a “Processing” widget, there should be another widget that shows the “Avg Log Size” in KiB.
